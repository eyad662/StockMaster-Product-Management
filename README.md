This is a complete **CRUD (Create, Read, Update, Delete)** application designed for basic stock and product management. The system is built purely with **HTML, CSS, and vanilla JavaScript**, making it fast, lightweight, and easy to understand.

## ‚ú® Key Features

* **Price Calculation:** Automatically calculates the final product price based on (Price + Taxes + Ads - Discount).
* **Local Storage Integration:** Persists product data in the browser's Local Storage, ensuring data is saved even after closing the browser.
* **Bulk Creation:** Allows adding multiple copies of the same product using the "Number of Items" (`count`) input.
* **Dynamic Search:** Search functionality by either **Product Title** or **Category**.
* **Responsive Design:** Optimized layout for seamless use on desktop, tablet, and mobile devices.
* **Update & Delete:** Full CRUD capabilities for managing individual product items.
